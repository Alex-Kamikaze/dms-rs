# translateRust

```
Utility for auto-generating JSON dictionaries with wide range of translating APIs

Usage: cli.exe <COMMAND>

Commands:

  init       Initialize new dictionaries system
  help       Print this message or the help of the given subcommand(s)

Options:
  -h, --help     Print help
  -V, --version  Print version
```
